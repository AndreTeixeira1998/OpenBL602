1. Developer Environment¶

此目录介绍编译或下载等工具的使用。

1.1. 编译¶

用户可以使用Windows平台进行编译，具体使用方法参考 Windows环境编译

1.2. 下载¶

用户可以使用BL自主开发的工具 BLFlashEnv 进行下载，具体使用方法参考 BLFlashEnv

1.3. Debug¶

选择 Freedom Studio + OpenOCD 方式进行debug。具体调试方法参考 Freedom Studio 。
